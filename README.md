# Mobile-Networks
This repository contains the Mobile Networks Laboratory experiments focused on developing Android applications using Java and Android Studio. These experiments cover fundamental Android components such as Views, Intents, Services, Broadcast Receivers, SQLite Database, Messaging, and Location-Based Services.

🛠 Technologies Used

Java
Android Studio
Android SDK
SQLite Database
XML (User Interface Design)

🧪 List of Experiments
1️⃣ Developing Simple Android Applications using Views and ViewGroups

Created basic UI using TextView, EditText, Button, LinearLayout, and RelativeLayout.

2️⃣ Developing Android Application using Menu

Implemented Options Menu and handled menu item selection.

3️⃣ Developing Mobile Application using Implicit Intent

Created application to perform actions like opening browser, calling, and sharing data using implicit intent.

4️⃣ Developing Android Application using Explicit Intent

Navigated between activities and passed data using explicit intent.

5️⃣ Developing Android Application using SQLite Database

Performed database operations such as insert, update, delete, and retrieve data.

6️⃣ Developing Messaging Services in Android

Developed application for sending and receiving SMS.

7️⃣ Building Location-Based Services in Android

Created application to get and display device location using GPS.

8️⃣ Developing Android Application using Broadcast Receivers

Implemented broadcast receiver to respond to system events.

9️⃣ Developing Android Application using Services

Created background service to perform tasks without user interaction.

🔟 Build an Application to Draw Basic Graphical Primitives

Developed application to draw shapes like Line, Circle, and Rectangle using Canvas.
